# Report Columns

Comprehensive guide to understanding and utilizing available data columns in Affelios reporting.

## Standard Columns

### Identification Columns
1. **Affiliate Information**
   - `affiliate_id` - Unique affiliate identifier
   - `affiliate_name` - Affiliate display name
   - `affiliate_email` - Affiliate contact email
   - `affiliate_status` - Active, inactive, or suspended
   - `affiliate_tier` - Performance tier classification

2. **Program Information**
   - `program_id` - Unique program identifier
   - `program_name` - Program display name
   - `operator_id` - Program operator identifier
   - `operator_name` - Operator company name
   - `program_status` - Active, paused, or terminated

3. **Campaign Information**
   - `campaign_id` - Campaign identifier
   - `campaign_name` - Campaign display name
   - `traffic_source` - Traffic source category
   - `creative_id` - Creative asset identifier
   - `link_id` - Tracking link identifier

### Date and Time Columns
1. **Transaction Timestamps**
   - `click_date` - Date and time of click
   - `conversion_date` - Date and time of conversion
   - `payment_date` - Date of commission payment
   - `created_date` - Record creation timestamp
   - `modified_date` - Last modification timestamp

2. **Period Aggregations**
   - `date` - Aggregation date (daily reports)
   - `week` - Week beginning date
   - `month` - Month and year
   - `quarter` - Quarterly period
   - `year` - Annual period

### Performance Metrics
1. **Traffic Metrics**
   - `clicks` - Total number of clicks
   - `unique_clicks` - Unique visitor clicks
   - `impressions` - Creative impressions
   - `ctr` - Click-through rate percentage
   - `bounce_rate` - Bounce rate percentage

2. **Conversion Metrics**
   - `conversions` - Total conversion count
   - `conversion_rate` - Conversion percentage
   - `first_conversions` - New customer conversions
   - `repeat_conversions` - Returning customer conversions
   - `conversion_value` - Total conversion value

### Financial Columns
1. **Revenue Data**
   - `gross_sales` - Total sales amount
   - `net_sales` - Sales after returns/refunds
   - `average_order_value` - Mean order value
   - `total_revenue` - Accumulated revenue
   - `refunds` - Refund amounts

2. **Commission Data**
   - `commission_rate` - Commission percentage/amount
   - `gross_commission` - Commission before deductions
   - `net_commission` - Commission after deductions
   - `paid_commission` - Actually paid commission
   - `pending_commission` - Unpaid commission amount

## Geographic Columns

### Location Data
1. **Geographic Information**
   - `country` - Country name or code
   - `country_code` - ISO country code
   - `region` - State/province/region
   - `city` - City name
   - `postal_code` - ZIP/postal code

2. **Geographic Aggregations**
   - `continent` - Continental grouping
   - `timezone` - Time zone identifier
   - `language` - Preferred language
   - `currency` - Local currency code
   - `exchange_rate` - Currency conversion rate

### Device and Technology
1. **Device Information**
   - `device_type` - Desktop, mobile, tablet
   - `operating_system` - OS name and version
   - `browser` - Browser name and version
   - `screen_resolution` - Display resolution
   - `mobile_brand` - Mobile device brand

2. **Technology Tracking**
   - `user_agent` - Full user agent string
   - `ip_address` - Visitor IP address (anonymized)
   - `referrer` - Referring page URL
   - `landing_page` - Destination page URL
   - `exit_page` - Last page visited

## Customer Data Columns

### Customer Information
1. **Customer Identification**
   - `customer_id` - Unique customer identifier
   - `customer_email` - Customer email (hashed)
   - `customer_type` - New or returning customer
   - `customer_segment` - Customer categorization
   - `lifetime_value` - Customer lifetime value

2. **Purchase Behavior**
   - `order_id` - Unique order identifier
   - `product_id` - Product identifier
   - `product_category` - Product category
   - `quantity` - Items purchased
   - `discount_amount` - Applied discounts

### Transaction Details
1. **Order Information**
   - `order_status` - Completed, pending, cancelled
   - `payment_method` - Payment type used
   - `shipping_cost` - Shipping charges
   - `tax_amount` - Tax charges
   - `coupon_code` - Applied coupon codes

2. **Fulfillment Data**
   - `shipping_date` - Order shipment date
   - `delivery_date` - Delivery completion date
   - `return_date` - Return processing date
   - `return_reason` - Reason for return
   - `refund_amount` - Refunded amount

## Custom Columns

### Configurable Fields
1. **Custom Parameters**
   - `custom_param_1` through `custom_param_10`
   - `sub_id` - Affiliate sub-identifier
   - `source_id` - Traffic source identifier
   - `medium` - Marketing medium
   - `content` - Content identifier

2. **Calculated Fields**
   - `epc` - Earnings per click
   - `roas` - Return on ad spend
   - `ltv_ratio` - Lifetime value ratio
   - `quality_score` - Traffic quality rating
   - `fraud_score` - Fraud risk assessment

### Integration Fields
1. **Third-Party Data**
   - `google_analytics_id` - GA transaction ID
   - `facebook_pixel_id` - Facebook pixel identifier
   - `crm_contact_id` - CRM system contact ID
   - `email_campaign_id` - Email campaign identifier
   - `social_media_id` - Social platform identifier

2. **Attribution Data**
   - `first_touch_source` - First interaction source
   - `last_touch_source` - Last interaction source
   - `assist_touches` - Intermediate touchpoints
   - `touch_count` - Total touchpoint count
   - `attribution_model` - Applied attribution model

## Data Types and Formats

### Numeric Formats
1. **Integer Values**
   - Whole numbers (clicks, conversions)
   - Counts and quantities
   - ID fields and identifiers
   - Status codes

2. **Decimal Values**
   - Percentages (rates, ratios)
   - Currency amounts
   - Calculated metrics
   - Score values

### Text Formats
1. **String Data**
   - Names and descriptions
   - URLs and domains
   - Categories and tags
   - Status descriptions

2. **Date Formats**
   - ISO 8601 format (YYYY-MM-DD)
   - Timestamp format (YYYY-MM-DD HH:MM:SS)
   - Unix timestamp
   - Relative date formats
